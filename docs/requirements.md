# Overview

Personal learning project for AI-assisted development.

This application helps dog owners track:

* daily health condition
* meals
* medications
* symptoms
* vet visits

The project focuses on:

* maintainable architecture
* simple implementation
* explicit frontend/backend separation
* Docker-based development

---

# Technical Constraints

* Vue 3
* FastAPI
* PostgreSQL
* Docker Compose

---

# Architecture Principles

* maintainability over scalability
* avoid over abstraction
* MVP first

---

# Current MVP Scope

## Goal

foundation setup

## Included

* initial setup
* connect frontend,  backend and db

## Not Included

* dog profile CRUD
* daily health logs
* recent history view
* notifications
* analytics
* AI diagnosis
* multi-user support
